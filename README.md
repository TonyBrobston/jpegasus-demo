# jpegasus-demo
This repo is the most simplistic implementation of Jpegasus.

## Jpegasus links:
* [Demo](https://tonybrobston.github.io/jpegasus-demo)

## To run locally:
`yarn start` and then navigate to http://localhost:5000/
